# libnomp

Runtime for [nomp](https://github.com/nomp-org/nomp)

## Build instructions

Clone the repo first:
```bash
git clone https://github.com/nomp-org/libnomp.git
cd libnomp
```

Use `cmake` to build the repo:
```bash
mkdir build
cd build
cmake ..
make
```

## Run tests

After building, you can run tests in tests/ directory.
```bash
cd tests
./run-tests.sh
```
