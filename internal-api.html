Backend init functions

int opencl_init(struct backend *backend, const int platform_id, const int device_id)

Initializes OpenCL backend with the specified platform and device.

Initializes OpenCL backend while creating the command queue using the given platform id and device id. Returns a negative value if an error occurs during the initialization, otherwise returns 0.

Parameters

  • backend – Target backend for code generation.

  • platform_id – Target platform id to share resources and execute kernals in the targeted device.

  • device_id – Target device id to execute kernals.

Returns

int
